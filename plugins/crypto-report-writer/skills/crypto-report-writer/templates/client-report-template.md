# Client Report Template

## Executive Summary

Summarize the issue, business impact, and remediation urgency in client-facing language.

## Finding Title

`[Severity] Short finding name`

## Severity and Impact

- Severity: Critical / High / Medium / Low
- Security property affected: soundness / privacy / integrity / authenticity / availability
- Practical impact on the client deployment

## Affected Scope

- Systems, modules, and deployments impacted
- Preconditions required for exploitation

## Root Cause

Describe the exact enforcement gap in implementation terms.

## Evidence Summary

- Verified trigger condition
- Reproduction status (PoC / executable proof / structured test evidence)
- Confidence level and validation notes

## Remediation Plan

- Concrete remediation action(s)
- Priority order and rollout recommendation
- Validation plan after fix (regression tests, negative tests, monitoring)

## Client Action Checklist

- [ ] Patch implemented
- [ ] Backports assessed
- [ ] Verification tests rerun
- [ ] Deployment communication prepared

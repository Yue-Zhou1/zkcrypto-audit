# Public Disclosure Template

## Advisory Title

`[Severity] Short finding name`

## Affected Versions

- Affected version range(s)
- First fixed version
- Mitigated but unfixed configurations (if any)

## Disclosure Timeline

- Date reported
- Vendor acknowledgment date
- Fix release date
- Public disclosure date

## Technical Summary

Concise explanation of the vulnerability and the security property affected.

## Impact

Explain potential attacker outcomes without exposing private client context.

## Remediation

- Upgrade guidance
- Workarounds (if immediate upgrade is unavailable)

## Credits

- Discoverer(s)
- Review contributor(s)
- Coordinating organization (if applicable)

## References

- Public commit(s) / advisory links
- CVE identifier(s) if assigned
- Reproducible public test evidence where appropriate
